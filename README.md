## 💡 市場の傾向・考察

1.  **大手アプリ vs 特化型アプリ:**
    「スタディサプリ」のような総合学習プラットフォームが圧倒的なシェアを持っていますが、「図形（面積クイズ）」や「計算（数学クイズ なん度？）」に特化したニッチなアプリも非常に高い評価とレビュー数を獲得しており、需要があることがわかります。

2.  **暗記系アプリの強さ:**
    社会、理科、漢字といった暗記科目向けの「一問一答」形式のアプリが多数ランクインしており、スキマ時間での学習ツールとしての地位を確立しています。

3.  **プラットフォーム間の差異:**
    iOS（App Store）における学習系アプリへのレビュー数がAndroidと比較して桁違いに多く、中学受験層（および保護者層）におけるiPhone/iPad利用率の高さが伺えます。
